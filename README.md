# terraform-module-repository-template

> [!CAUTION]**Do not publish this repository.**
> 
> This repository is a private repository template for repositories to be created within the [Terraform Modules for BTP outbound request](https://github.tools.sap/OSS-Outbound-Process/terraform-btp-modules-multiple-).

## About this project

Template repository for BTP Terraform modules based on the [Terraform Provider for SAP BTP](https://registry.terraform.io/providers/SAP/btp/latest/docs).
